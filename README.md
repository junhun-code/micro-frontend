# micro-frontend
基于single-spa框架拓展多个vue项目的微前端实践


## 安装步骤：
### 1. 主应用（master）：
    - cd master-app
    - npm install / cnpm install
    - npm run dev
### 2. 子应用（about）：
    - cd sub-app-about
    - npm install / cnpm install
    - npm run dev
### 3. 子应用（home）：
    - cd sub-app-home
    - npm install / cnpm install
    - npm run dev
